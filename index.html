About the Project

This project explores playful and human-centric ways to support group discussions at XJTLU. Our target users include XJTLU students, language teachers, and academic teachers. The system is intended for mobile and PC use in onsite discussion contexts.

Group discussion is an important part of university learning. It affects participation, confidence, communication, and collaboration. However, current discussion activities may still involve problems such as unequal participation, unclear structure, awkward silences, and limited support for teachers to monitor group progress.

This project mainly involves XJTLU students, language teachers, and academic teachers, and the project is intended for mobile or PC use in an onsite context.

Initial Brainstorming

Personas

Persona 1 – Student

Goal: Participate confidently in discussions.

Pain Point: Feels nervous and does not know when to speak.

Persona 2 – Teacher

Goal: Encourage balanced and meaningful participation.

Pain Point: Hard to monitor engagement and discussion quality.

User Journey Map

Early Prototype Ideas

Idea 1 - Discussion Prompt Support Tool

A mobile or web-based tool that provides students with prompts to help them start, continue, and deepen group discussions.

Idea 2 - Participation Balance Assistant

A lightweight interactive system that helps visualise speaking balance in a group and encourages more equal participation.

Idea 3 - Teacher Facilitation Dashboard

A simple tool that allows teachers to monitor group progress, identify quieter groups, and provide timely support during discussions.
